# Contributing to CodeDoc

Thank you for wanting to improve CodeDoc! To keep the project stable, please follow these steps:

1. **Fork the repo** and create your branch from `main`.
2. **Run your code** locally using `uv run server.py` to ensure it starts without errors.
3. **Open a Pull Request (PR)** describing your changes.
4. **Wait for Audit:** All PRs must pass the automated linting checks and be manually approved by @akshay1018.

Note: Direct pushes to `main` are disabled.
